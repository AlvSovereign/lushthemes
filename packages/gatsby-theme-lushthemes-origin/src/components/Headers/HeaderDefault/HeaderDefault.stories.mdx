# HeaderDefault

THe meat of the Header. This component contains the logo, nav links and any other CTAs you'd want to add.

IMPORTANT: It receives 2 props from the parent component by way of 'child cloning', they are:
- showMenu: a boolean that controls the displa of the modal menu
- setShowMenu: a React.SetStateAction function that toggles the showMenu function.
`showMenu` and `setShowMenu` are React state hooks. You can use this to toggle and menu added to the `<HeaderContainer />` component from this `<HeaderDefault />` component.
