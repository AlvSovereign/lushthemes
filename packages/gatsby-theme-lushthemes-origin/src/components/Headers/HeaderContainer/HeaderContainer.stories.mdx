# HeaderContainer

The wrapper of the container controlling how the container behaves when the user scrolls. It also takes a `ResponsiveMenu` which is a menu to display for mobile & tablet screen sizes. It also clones the children of this compoent to add 2 props (a React state hook) that is responsible for toggling the `responsiveMenu`.
